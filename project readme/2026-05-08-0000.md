## 页面路由与导航

| 页面路径 | tabbar | 说明 |
|----------|--------|------|
| pages/index/index | 是（tab 0） | "数据"标签，数据输入 + 跳转到展示页 |
| pages/profile/profile | 是（tab 1） | "我的"标签，微信登录 + 用户信息展示 |
| pages/show/show | 否 | 数据列表展示，从 index 通过 navigateTo 进入 |

- tabbar 采用 `custom: true` 自定义组件方式实现，代码在 `custom-tab-bar/`
- 每个 tab 页面在 `onShow` 中通过 `this.getTabBar().setData({ selected: N })` 设置选中态
- 非 tab 页面不会显示底部导航栏

## 云函数 API（quickstartFunctions）

通过 `event.type` 字段区分操作：

| type | 功能 | 参数 | 涉及数据库 |
|------|------|------|-----------|
| addUserData | 写入用户数据 | event.content | userData 集合 |
| getUserData | 查询所有数据（按时间倒序） | 无 | userData 集合 |
| getOpenId | 获取用户 openid | 无 | - |
| getPhoneNumber | 解密手机号 | event.code | - |
| getMiniProgramCode | 生成小程序码 | 无 | - |
| createCollection | 创建 sales 集合 | 无 | sales 集合 |
| selectRecord | 查询 sales 记录 | 无 | sales 集合 |
| updateRecord | 更新 sales 记录 | event.data | sales 集合 |
| insertRecord | 新增 sales 记录 | event.data | sales 集合 |
| deleteRecord | 删除 sales 记录 | event.data | sales 集合 |

### 数据库集合

**userData**（当前项目主要使用的集合）:
| 字段 | 类型 | 说明 |
|------|------|------|
| _id | string | 自动生成 |
| userId | string | 用户 openid |
| inputTime | string | 输入时间（北京时间） |
| inputContent | string | 用户输入内容 |

**sales**（旧 demo 数据）:
| 字段 | 说明 |
|------|------|
| region | 地区 |
| city | 城市 |
| sales | 销量 |

## 页面功能详情

### 首页 - 数据输入（pages/index/index）

- 标题："数据"
- 输入框 + 绿色的"提交"按钮
- 提交调用 `addUserData`，写入 userId、inputTime、inputContent
- 底部蓝色链接"查看已提交数据 →"，跳转到 show 页面

### 数据展示页（pages/show/show）

- 标题："数据列表"
- `onShow` 时自动调用 `getUserData` 拉取数据
- 卡片列表展示每条记录的 userId、inputTime、inputContent
- 无数据时显示"暂无数据"

### 个人中心（pages/profile/profile）

- 顶部一个白色圆角长方形 box（140rpx 高）
- **未登录**：box 居中显示绿色"微信登录"按钮
- **登录后**：box 左侧圆形头像（80rpx）+ 右侧微信昵称
- 登录后 box 下方显示红色"退出登录"按钮

## 用户强调过的要求

### 登录方式
- **必须用 `open-type="getUserInfo"`**，不能用 `wx.getUserProfile`（已失效，不会弹出授权框）
- `<button open-type="getUserInfo" bindgetuserinfo="onWxLogin">微信登录</button>`
- 回调中通过 `e.detail.userInfo` 获取 avatarUrl 和 nickName
- **不要**单独的 `chooseAvatar` + `nickname` 输入方式，就要一个按钮弹出微信授权

### UI 细节
- 登录后只需展示头像 + 昵称，**不需要**展示手机号
- 按钮文字居中：当 button 有默认 padding 时，用 `display: flex; align-items: center; justify-content: center; padding: 0;` 代替 `line-height: height`
- 只用 `line-height: height` 居中文字的前提是**垂直 padding 为 0**

### 开发习惯
- 不要随意修改已有的页面样式，除非明确指出样式有问题
- 修改代码聚焦在用户真正提出的问题上，不要顺手改其他东西

### 云函数
- 本地调试时操作的是**真实云端数据库**，不是本地数据库
- 上传云函数推荐用"云端安装依赖（不上传 node_modules）"
